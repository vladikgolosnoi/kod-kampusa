# Код Кампуса

«Код Кампуса» — самостоятельная игра из пяти букв для дня открытых дверей IT-вуза. Игроку нужно угадывать русские слова на тему учёбы, технологий, лабораторий и студенческой жизни.

## Что внутри

- русский интерфейс и русская экранная клавиатура;
- тематический словарь под IT-вуз;
- непрерывный режим: после ответа можно сразу перейти к следующему слову;
- статистика, шаринг результата и пользовательские настройки.

## Локальный запуск

```bash
npm ci
npm start
```

После запуска приложение обычно доступно на `http://localhost:3000/`.

## Основные скрипты

```bash
npm start     # dev-сервер
npm test      # тесты
npm run build # production build
```

## Структура

- `src/constants/wordlist.ts` — слова для раундов.
- `src/constants/validGuesses.ts` — допустимые догадки.
- `src/App.tsx` — игровой цикл, переход между раундами и модалки.

## Идея проекта

Проект оформлен как отдельная игра для мероприятия, а не как форк публичного демо. При необходимости можно дальше доработать словарь, визуальный стиль и контент под конкретный вуз или факультет.
